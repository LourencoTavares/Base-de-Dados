# Base de Dados

Projetos desenvolvidos no âmbito da UC de **Base de Dados**.

O trabalho foi dividido em duas entregas, começando pela modelação conceptual e relacional de um sistema para um jardim zoológico, e terminando com a implementação da base de dados, restrições de integridade, consultas SQL, análise de dados e desenvolvimento de uma API REST.

---

## Descrição Geral

O projeto teve como objetivo desenvolver uma solução de base de dados para um sistema de informação de um **jardim zoológico**.

Ao longo das duas entregas foram abordadas várias fases do desenvolvimento de uma base de dados:

- análise do domínio;
- modelação Entidade-Associação;
- conversão para o modelo relacional;
- definição de Restrições de Integridade;
- normalização;
- implementação em SQL/PostgreSQL;
- povoamento da base de dados;
- desenvolvimento de consultas SQL;
- criação de vistas e índices;
- implementação de uma API REST para interação com a base de dados.
